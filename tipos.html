Tipos de datos de Big Data

Big Data abarca datos de diferentes tipos y estructuras. Para comprender mejor su naturaleza, se clasifican en tres categorías principales: estructurados, semiestructurados y no estructurados.

Datos estructurados

Los datos estructurados se distinguen por tener una organización definida y predecible. Se organizan en un formato fijo, como filas y columnas en una tabla o base de datos. Suelen ser fácilmente almacenados y procesados por sistemas tradicionales de bases de datos.

  • Ejemplos de datos estructurados:
    • Registros de transacciones
    • Datos de sensores
    • Datos de redes sociales en formato tabular
    • Datos de censos y encuestas
    • Información financiera

Datos semiestructurados

Los datos semiestructurados poseen cierta organización, pero no se ajustan a un formato fijo o rígido. Aunque no siguen un esquema estricto, contienen elementos que permiten identificar la información que representan.

  • Ejemplos de datos semiestructurados:
    • Archivos JSON
    • Archivos XML
    • Correos electrónicos
    • Datos de logs
    • Datos de sensores con formato flexible

Datos no estructurados

Los datos no estructurados carecen de una organización o estructura predefinida. A menudo se presentan en formatos humanos comprensibles, pero complejos para la interpretación directa por un sistema informático. Suelen requerir técnicas de procesamiento de lenguaje natural y aprendizaje automático para su análisis.

  • Ejemplos de datos no estructurados:
    • Texto libre (documentos, libros, artículos, redes sociales)
    • Imágenes (fotografías, gráficos, imágenes médicas)
    • Audio (grabaciones de voz, música)
    • Video (películas, videos de vigilancia, videos en redes sociales)
